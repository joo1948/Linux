## 1. 웹 서비스 메소드 설정 공격
- 설명 : 웹 서비스 제공 시 불필요한 메소드(PUT, DELETE 등) 허용으로 공격자에 의해 악성파일을 업로드 하거나 중요 파일 삭제가 가능해지는 취약점.

### 해결방법
- httpd.conf & httpd-ssl.conf에 모두 적용
- 
``` shell
<Location />
   <LimitExcept GET POST>
     Order deny,allow
      Deny from all
    </LimitExcept>
</Location>

TraceEnable Off

```
